<h1 class="ap-hero__title">Pylectra</h1>

<p class="ap-hero__sub">A plugin-based Python framework for power-system dynamic simulation and large-scale dataset generation.</p>

<div class="ap-features" markdown>

<div class="ap-feature" markdown>
:material-puzzle:{ .ap-feature__icon style="font-size:36px" }
<p class="ap-feature__label">Plugin-first</p>
<p class="ap-feature__caption">Generators, solvers, faults, plots — every component is a registered plugin you swap in one line of YAML.</p>
</div>

<div class="ap-feature" markdown>
:material-flash:{ .ap-feature__icon style="font-size:36px" }
<p class="ap-feature__label">GPU-ready</p>
<p class="ap-feature__caption">Six ODE solvers including GPU-accelerated paths for batch generation at dataset scale.</p>
</div>
